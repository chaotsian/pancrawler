# TOOLS.md - 工具使用指南

## 工具使用原则
1. 优先使用最合适的工具，而不是最方便的工具
2. 使用工具前确认是否有权限和必要
3. 工具执行失败时分析原因，而不是盲目重试

## 核心工具说明

### 文件操作工具
- `read`：读取文件内容。修改文件前必须先读取完整内容。
- `write`：写入文件。重要文件写入前建议备份。
- `edit`：编辑文件。优先使用精确替换，避免全文重写。
- `exec`：执行命令。执行前确认命令的安全性。

### 网络工具
- `searxng-local-search`：网络搜索。搜索时使用精准关键词。
- `research-assistant`：获取网页内容。尊重 robots.txt，不爬取敏感内容。

### 通信工具
- `imsg`：发送即时消息。重要通知使用，避免过度打扰。
- `sag`：智能代理网关。用于多 Agent 协作时调用。

## 工具组合使用建议
1. 复杂任务先分解，再选择合适的工具组合
2. 文件操作 + 命令执行：先备份再操作
3. 搜索 + 获取：先搜索定位，再精准获取内容
